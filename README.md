# Customer Transaction Prediction

## Project Overview

Customer Transaction Prediction is a machine learning classification problem that aims to determine whether a customer will perform a financial transaction in the future based on historical data.
## Objective

- Develop a classification model to predict customer transactions
  
- Explore and analyze customer-related features
  
- Compare different machine learning algorithms

- Identify the most effective model
## Dataset

- Consists of anonymized customer information
  
- Includes multiple numerical input features

## Target Variable:

- 0 → No Transaction
 
- 1 → Transaction Occurred

## Models Implemented

- Logistic Regression

- Random Forest

- Gradient Boosting

- XGBoost
## Evaluation Metrics

- Accuracy

- Precision

- Recall

- F1 Score

- ROC-AUC Score

## Best Performing Model

- XGBoost delivered the best results with superior ROC-AUC and recall values
